## Installazione

Prerequisiti:
- OCaml (con anche ocamlc)
- Python 3.x
    - pyvis (`pip install pyvis`)

Per compilare il progetto eseguire i seguenti comandi:

```bash
cd progetto
make
```

E' possibile ripulire dall'esecuzione di `make` con: `make clear`.
